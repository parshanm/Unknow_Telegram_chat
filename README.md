
# Unknow_Telegram_Chat

You can chat with someone Unknow!!!
You will connect to a random person and you can chat with them. 

## Installation

How to run this robot

first you must install the libs:

```bash
pip install pyTelegramBotAPI
pip install sqlite3
```
run the main file:
```bash
python3 main.py
```
